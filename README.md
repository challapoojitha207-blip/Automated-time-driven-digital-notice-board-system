# Automated Time-Driven Digital Notice Board System

## 📌 Project Overview

The **Automated Time-Driven Digital Notice Board System** is a web-based application designed to replace traditional physical notice boards used in colleges, offices, and apartments.
It allows users to create, manage, and display notices digitally with details such as title, description, priority level, and expiry time.

The system automatically removes expired notices to ensure that only active and relevant information is displayed.

---

## 🎯 Features

* Create and publish notices with title, description, priority, and expiry time
* Automatically remove expired notices
* Sort notices based on priority and time
* Search and filter notices easily
* User-friendly and responsive interface
* Channel-based notice management

---

## 🚀 How to Use

1. Open the website in a browser.
2. Create a new notice using the notice form.
3. Enter the title, description, priority level, and expiry time.
4. Submit the notice to display it on the notice board.
5. Notices will automatically disappear after the expiry time.
